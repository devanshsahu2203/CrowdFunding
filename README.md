This project focuses on analyzing crowdfunding campaign data to understand the key factors that contribute to successful fundraising campaigns. Crowdfunding platforms allow creators, entrepreneurs, and organizations to raise funds from a large number of people, but not all campaigns achieve their funding goals. The purpose of this project is to explore campaign data and identify patterns that influence campaign success and funding performance.

The analysis involves working with a crowdfunding dataset containing information such as campaign categories, funding goals, pledged amounts, number of backers, campaign duration, and campaign outcomes. Using this data, meaningful insights were extracted to better understand how different variables impact the success or failure of crowdfunding projects.

The project follows a complete data analysis workflow, starting from data cleaning and preparation to exploratory analysis and interactive dashboard creation. SQL was used to perform data extraction, transformation, and querying to generate key metrics. The processed data was then visualized using Power BI/Tableau, where an interactive dashboard was built to present insights in a clear and engaging way.

Key Objectives of the Project

Analyze overall crowdfunding performance and funding trends

Identify the success rate of campaigns across different categories

Compare campaign goals with the amount of money pledged by backers

Understand the relationship between the number of backers and funding success

Examine how campaign duration impacts funding outcomes

Identify top-performing categories and funding patterns

Key KPIs Analyzed

Total number of campaigns

Total amount pledged

Total number of backers

Successful vs failed campaigns

Average funding per campaign

Funding goal vs pledged amount comparison

Dashboard Insights

The dashboard highlights important insights such as:

Certain categories consistently receive higher funding compared to others.

Campaigns with realistic funding goals tend to have a higher success rate.

A higher number of backers strongly correlates with successful campaigns.

Some campaign durations show better funding performance than others.

Tools and Technologies Used

SQL – Data extraction, transformation, and analysis

Power BI / Tableau – Data visualization and dashboard creation

Excel / Data preprocessing – Data cleaning and preparation

The final interactive dashboard provides a clear overview of crowdfunding trends and helps identify the factors that drive successful fundraising campaigns.
